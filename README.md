# HexGridGenerator
A Hex Grid Generator that can spawn color coded tiles representing different terrain types. 

## Features
- 5 different terrain types - Grass, Water, Mountain, Dessert and Snow
- Different terrain heights for Water and Mountains
- Random Terrain Generation

## Technologies
- Unreal Engine 5
- C++
- Blueprints
